# Prompt Siêu Ngắn

```text
[Spec]
Viết spec cho: ...
Bối cảnh: ...
Đầu ra: tổng quan, mục tiêu, use case, yêu cầu, edge case, acceptance criteria.

[Điều tra]
Điều tra lỗi: ...
Hiện tượng: ...
Log: ...
Đã thử: ...

[Fix bug]
Fix bug: ...
Hiện tại: ...
Mong muốn: ...
Hãy tìm root cause, sửa, và đưa test cases.

[Review]
Review code này: ...
Ưu tiên bug, regression risk, performance, security, edge cases, missing tests.

[Code]
Viết code cho: ...
Bối cảnh: ...
Hãy phân tích ngắn, viết code, và đề xuất test cases.
```
