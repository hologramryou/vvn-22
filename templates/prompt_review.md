# Prompt Review Code

```text
Vai trò: bạn là senior engineer review code.
Mục tiêu: review đoạn code / PR sau: ...
Bối cảnh:
- Mục tiêu thay đổi: ...
- File/module chính: ...
- Ràng buộc hệ thống: ...
Đầu ra mong muốn:
- Findings trước
- Mỗi finding ghi rõ:
  - Mức độ: high / medium / low
  - Vấn đề là gì
  - Vì sao đáng lo
  - Cách sửa đề xuất
- Nếu không có issue nghiêm trọng, nói rõ điều đó
Ràng buộc:
- Tập trung vào vấn đề thực chất
- Không nitpick style nếu không ảnh hưởng hành vi
```
