# Prompt Viết Spec

```text
Vai trò: bạn là senior product engineer.
Mục tiêu: viết spec cho tính năng sau: ...
Bối cảnh:
- Sản phẩm/hệ thống: ...
- Người dùng liên quan: ...
- Vấn đề cần giải quyết: ...
- Phạm vi: ...
- Ngoài phạm vi: ...
Đầu ra mong muốn:
- Tổng quan
- Mục tiêu
- User flow / use case
- Yêu cầu chức năng
- Yêu cầu phi chức năng
- Edge cases
- Acceptance criteria
- Rủi ro / câu hỏi mở
Ràng buộc:
- Viết ngắn gọn nhưng đủ để implement
- Format markdown
```
