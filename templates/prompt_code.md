# Prompt Viết Code

```text
Vai trò: bạn là senior software engineer.
Mục tiêu: viết code cho yêu cầu sau: ...
Bối cảnh:
- Ngôn ngữ/framework: ...
- Kiến trúc/project structure: ...
- File liên quan: ...
- Input/output mong muốn: ...
- Convention của dự án: ...
Đầu ra mong muốn:
- Phân tích ngắn cách làm
- Code hoàn chỉnh
- Giải thích phần quan trọng
- Test cases
- Assumption nếu có
Ràng buộc:
- Code rõ ràng, dễ maintain
- Không thêm complexity không cần thiết
- Giữ đúng style hiện có của project
```
